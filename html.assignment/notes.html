Five html tags

li tag
The li tag is used to start a list, it defines what the list is.
br tag
Br stands for break tag and it is used to end a line and start on the next one.
hr tag
The abbreviation hr stands for horizontal rule. Its function is to create a horizontal line between two different subjects, for example:paragraphs.
blockquote tag
The blockquote tag is used to define a section that is quoted from another source
header tag
The header tag is used to define what the header is for the document/section.
